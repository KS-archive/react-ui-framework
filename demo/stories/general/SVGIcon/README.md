# SVGIcon

| Nazwa     | Typ               | Wartość domyślna | Opis                      |
|-----------|-------------------|------------------|---------------------------|
| **path** 	| string            | -		             | Ścieżka do pliku z ikoną. |
| className | string 	          | ```''```				 | Klasa dodana do ikony.	   |
| fill      | string 	          | ```''``` 		     | Kolor wypełnienia.        |
| height 	  | string / number   | ```''``` 		     | Wysokość.	               |
| width 		| string / number   | ```''``` 	       | Szerokość.                |
